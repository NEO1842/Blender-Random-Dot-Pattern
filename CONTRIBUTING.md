# Contributing

Thank you for your interest in contributing!

## Ways to Contribute

- Improve Geometry Nodes setups
- Add new procedural patterns
- Improve documentation
- Share rendering ideas
- Optimize node performance

## Workflow

1. Fork the repository
2. Create a branch
3. Make your changes
4. Commit your changes
5. Open a Pull Request

## Guidelines

- Keep node setups organized
- Use clear naming
- Avoid unnecessary complexity
- Keep files clean and readable
