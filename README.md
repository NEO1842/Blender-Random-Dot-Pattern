# Blender-Random-Dot-Pattern

Blender using Geometry node and shader nodes, inspired by Ducky 3D.

# Overview

This project is a scene file for Generated by using a Geometry node. 

It is based on a tutorial by Ducky 3D, but has been customized with additional nodes and adjusted parameters to achieve a unique visual style and randomized dot patterns.

# Software Requirement

Blender Version3.4

# Rendering Settings

Engine: Eevee

Sampling: 64

Denoising: None

Original Concept: Inspired by Ducky 3D

# Blender ランダムドットパターン

Blenderでジオ​​メトリノードとシェーダーノードを使用。Ducky 3Dに触発された作品。

# 概要

このプロジェクトは、ジオメトリノードを使用して生成されたシーンファイルです。

これはDucky 3Dのチュートリアルをベースにしていますが、独自のビジュアルスタイルとランダムなドットパターンを実現するために、追加のノードと調整されたパラメーターでカスタマイズされています。

# ソフトウェア要件

Blender バージョン3.4

# レンダリング設定

エンジン: Eevee

サンプリング数：64

ノイズ除去：なし

オリジナルコンセプト：Ducky 3Dにインスパイアされた
